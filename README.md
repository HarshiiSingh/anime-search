# AnimeSearch

This is the Final Project for TripleTen, it is a custom made website that utilizes a third party API to help the user with finding different anime to watch.

## Project Features

- React
- Vite.js
- JavaScript
- JikanAPI
- Dynamic Grid Layout
- Positioning
